### STEP 2: Deploy Your Website (30 minutes)

Choose ONE option:

#### Option A: Vercel (Easiest - Recommended)
```bash
npm install -g vercel
cd frontend
vercel deploy --prod
# Follow prompts, get your URL
```

#### Option B: Netlify
```bash
npm install -g netlify-cli
cd frontend
npm run build
netlify deploy --prod --dir=dist
```

#### Option C: GitHub Pages
```bash
cd frontend
npm install --save-dev gh-pages
# Add to package.json:
"homepage": "https://Sarvesh-Ranjan-9065.github.io/my-portfolio",
"deploy": "gh-pages -d dist"
# Then:
npm run build
npm run deploy
```

### STEP 3: Submit Sitemap & Set Up Analytics (30 minutes)

1. **Google Analytics 4:**
   - Go to https://analytics.google.com/
   - Create property → Get Measurement ID
   - Update index.html line 147

2. **Google Search Console:**
   - Go to https://search.google.com/search-console/
   - Add property → Verify ownership
   - Submit sitemap: yoursite.com/sitemap.xml
   - Update index.html line 159

## 🎓 Understanding Each Component

### 1. What is robots.txt?
- Tells search engines which pages to crawl
- Located at: yoursite.com/robots.txt
- Points to your sitemap

### 2. What is sitemap.xml?
- Lists all pages on your site
- Helps search engines index your content
- Submit to Google Search Console

### 3. What is Structured Data?
- JSON-LD schema markup
- Helps search engines understand your content
- Can show rich results in search (profile cards, etc.)

### 4. What are Open Graph tags?
- Meta tags for social media sharing
- Controls how your site looks when shared on Facebook/LinkedIn
- Includes title, description, image preview

### 5. What is Google Analytics 4?
- Tracks website visitors and behavior
- Shows: traffic sources, page views, conversions
- Free tool from Google

### 6. What is Google Search Console?
- Shows how your site appears in Google Search
- Tracks: keywords, rankings, indexing status
- Free tool from Google
