## 9. Semantic Keywords & LSI Keywords

### Related Terms (Latent Semantic Indexing)
- **For "Backend Developer":** server-side, API development, database management, authentication, scalability
- **For "DevOps Engineer":** automation, continuous integration, deployment, infrastructure, monitoring
- **For "Golang":** Go programming, concurrency, performance, goroutines
- **For "Docker":** containerization, images, docker-compose, orchestration
- **For "Kubernetes":** k8s, pods, deployments, clusters, helm

### Usage in Content
```
Instead of: "I am a backend developer with Golang experience."
Write: "I am a backend developer specializing in server-side API development using
Golang, with expertise in building scalable microservices, implementing authentication
systems, and managing database architectures."
```

## 11. Keyword Density Guidelines

### Recommended Density
- **Primary Keyword (per page):** 2-3% (backend developer, devops engineer)
- **Secondary Keywords:** 1-2% each
- **LSI Keywords:** Natural usage throughout content

### Example (1000-word About page)
- "backend developer" or "backend engineer": 20-30 times
- "DevOps engineer": 10-20 times
- "Golang" or "Go": 15-25 times
- Related terms (API, microservices, Docker, etc.): Natural frequency

### ⚠️ Warning
- Avoid keyword stuffing (>5% density)
- Maintain natural, readable content
- Focus on user experience over keyword density

## 16. Quick Reference: Keyword Placement

### Critical Locations (High Impact)
1. ✅ **Page Title** (`<title>` tag) - Primary keyword at the beginning
2. ✅ **Meta Description** - Primary + secondary keywords naturally
3. ✅ **H1 Heading** - Primary keyword
4. ✅ **H2 Headings** - Secondary keywords
5. ✅ **First Paragraph** - Primary keyword in first 100 words
6. ✅ **URL** (if multi-page) - Short, keyword-rich
7. ✅ **Alt Text** - Descriptive with keywords where relevant
8. ✅ **Internal Links** - Keyword-rich anchor text

### Important Locations (Medium Impact)
9. ✅ **H3-H6 Headings** - Long-tail keywords
10. ✅ **Bold/Strong Text** - Emphasize important keywords
11. ✅ **Image Filenames** - project-golang-api.jpg
12. ✅ **Schema Markup** - jobTitle, knowsAbout properties

### Supporting Locations (Lower Impact)
13. ✅ **Body Content** - Natural keyword usage
14. ✅ **Button Text** - "Hire Backend Developer"
15. ✅ **Navigation Menu** - Clear section names
16. ✅ **Footer** - Keywords in footer links
